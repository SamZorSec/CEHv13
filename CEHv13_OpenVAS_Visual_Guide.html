Installation on Kali Linux
Install & initialise
# Install OpenVAS / GVM sudo apt update && sudo apt install -y openvas # Run setup — downloads NVT feed (50 000+ checks). Takes 10–20 min. # ⚠ Save the admin password printed at the end! sudo gvm-setup # Verify all components are healthy sudo gvm-check-setup
Critical: gvm-setup prints the auto-generated admin password once at the very end of its output. Copy it immediately — there is no way to retrieve it later (only reset it).
Start / stop services
Service commands
# Start all GVM daemons sudo gvm-start # Stop all GVM daemons sudo gvm-stop # Three daemons that gvm-start launches: # ospd-openvas → scanner engine # gvmd → management daemon # gsad → web UI (HTTPS on port 9392) # Check individual daemon status sudo systemctl status ospd-openvas sudo systemctl status gvmd sudo systemctl status gsad # Restart scanner only (if "Scanner not connected") sudo systemctl restart ospd-openvas

Reliable minimum: ≥ 70%

Time management: In the 6-hour CEH practical, OpenVAS tasks take the longest wall-clock time (5–30 min per scan). Start your OpenVAS scan early, then switch to other tasks while it runs. Come back to analyse the report later — don't sit watching the progress bar.
