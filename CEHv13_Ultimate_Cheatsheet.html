Nmap essential commands
nmap -sS -sV -O <target> # SYN + version + OS detection nmap -sS -sV -A -T4 <target> # Aggressive (all-in-one) nmap -p- <target> # All 65535 ports nmap -p 22,80,443 <target> # Specific ports nmap -sU -p 53,161 <target> # UDP scan nmap -sn 192.168.1.0/24 # Ping sweep / host discovery nmap --script vuln <target> # Run vulnerability scripts nmap --script=banner <target> # Banner grab nmap -oN out.txt -oX out.xml <target> # Save output (normal + XML) nmap -D RND:10 <target> # Decoy scan (evasion) nmap -T0 --max-retries 1 <target> # Paranoid/stealthy

Hashcat — GPU-accelerated offline cracking
Hashcat — full reference
hashcat -m 0 hash.txt rockyou.txt # MD5 dictionary hashcat -m 1000 hash.txt rockyou.txt # NTLM dictionary hashcat -m 1800 hash.txt rockyou.txt # sha512crypt (Linux $6$) hashcat -m 3200 hash.txt rockyou.txt # bcrypt hashcat -m 22000 cap.hc22000 rockyou.txt # WPA2 # Attack modes (-a) hashcat -m 0 -a 0 hash.txt wordlist.txt # -a 0 dictionary hashcat -m 0 -a 1 hash.txt w1.txt w2.txt # -a 1 combination hashcat -m 0 -a 3 hash.txt ?u?l?l?l?d?d # -a 3 brute/mask hashcat -m 0 -a 6 hash.txt wordlist.txt ?d?d?d # -a 6 hybrid word+mask # Rules (mangling — fastest practical approach) hashcat -m 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule hashcat -m 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/dive.rule # Mask charset: ?l=lowercase ?u=uppercase ?d=digit ?s=special ?a=all # Key flags: -O optimised kernels --force ignore warnings (VM) # --show display cracked -o out.txt save cracked results # --status live progress --session name --restore resume

Step-by-step

Step 1 — Create target: Configuration → Targets → New Target

FieldValue
HostsIP address, range, or CIDR (e.g. 192.168.1.0/24)
Port listAll IANA assigned TCP (default)
Alive testSet "Consider Alive" if target blocks ping
CredentialsAdd SSH/SMB creds for authenticated scan

Step 2 — Create task: Scans → Tasks → New Task
Select target · Scanner = OpenVAS Default · Scan Config = Full and fast
Step 3 — Run: Click the green Play button → monitor % progress bar
Step 4 — Report: Scans → Reports → sort by Severity → export PDF / XML / CSV
